# Banking App - Biometric Face Login

A Python-based biometric authentication system using face recognition.
Built with OpenCV and MediaPipe.

## Features
- Real-time face detection via webcam
- Face registration system
- Face login with 92-96% accuracy match score
- Access granted/denied with confidence percentage

## How to run
```
python3.11 register_face.py  # register your face first
python3.11 login.py          # then run the login
```
